## 実証済み

実HTTP fixtureとChromiumで次を確認した。

- 可視・有効・一意な要素だけをcandidate化し、操作後に再観測して旧candidateを拒否する。
- iframe、popup、native dialogをadapter内部で追跡し、公開DTOへruntime handleを漏出しない。
- Observationから状態fingerprint、状態遷移グラフ、adaptive/replay-trace.jsonを生成する。
- traceのcandidate DTOをschemaで自動判別し、同じ安全制御を通してstrict replayする。
- generic oracleをproduct oracle拡張点から分離する。
- Airtest/PocoおよびSecurityは既存ツールを呼ぶbridgeとしてのみ接続し、DTO境界を検証する。

実行結果: npm run check は docs contract: pass、69 passed。
